## 説明

専門学生時代に制作した 2D アクションゲームを、ブラウザで遊べるように移植したバージョンです。

オリジナル版は C で制作した Windows 向けゲームで、このリポジトリでは当時の素材や挙動をもとに、Phaser を使ったブラウザ版として再実装しています。

## ゲーム概要

横スクロール型の 2D アクションゲームです。
プレイヤーを操作してステージを進み、敵やボスを倒しながらクリアを目指します。

## 使用言語・技術

- TypeScript
- HTML
- CSS
- Phaser
- Vite


## 起動方法

```bash
cd browser-game
npm install
npm run dev
```

起動後、表示されたローカルURLをブラウザで開いてください。

ビルドする場合:

```bash
npm run build
```

## 操作方法

| 操作 | キー / マウス |
| --- | --- |
| ゲーム開始 | Enter / クリック |
| 左右移動 | A / D |
| ジャンプ | Space |
| 歩き | B を押しながら移動 |
| 通常攻撃 | 左クリック |
| 攻撃チャージ | 左クリック長押し |
| 特殊攻撃 | 右クリック |
| 武器切り替え | Shift |
| 赤ポーション使用 | E |
| 青ポーション使用 | R |
| ゲームオーバー選択 | ↑ / ↓ |
| 決定 | Enter |

### 特殊攻撃の補足

- 剣モード中に `W` を押しながら右クリックすると、上方向の特殊攻撃を出します。
- 剣モード中に `S` を押しながら右クリックすると、空中回転系の特殊攻撃を出します。
- 矢モード中は、右クリックで複数の矢を放つ特殊攻撃を出します。
